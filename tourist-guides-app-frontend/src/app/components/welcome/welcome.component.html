Καλωσορίσατε στις Ψηφιακές Υπηρεσίες του Σωματείου Διπλωματούχων Ξεναγών

Μέσω των ψηφιακών υπηρεσιών μπορείτε:

  • Αν είστε Διπλωματούχος Ξεναγός
    • Να καταχωρήσετε τα στοιχεία σας στον Κατάλογο Διπλωματούχων Ξεναγών, ο οποίος είναι ορατός σε κάθε εγγεγραμμένο χρήστη της πλατφόρμας
    • Να ενημερώνεστε για τις δραστηριότητες που διοργανώνει το Σωματείο στα πλαίσια της συνεχούς εκπαίδευσης που παρέχει στα μέλη του (σεμινάρια, εκδρομές, επισκέψεις σε μουσεία και εκθέσεις κ.ά) και να αποθηκεύετε τις αγαπημένες σας
  • Αν είστε Επισκέπτης
    • Να έχετε πρόσβαση στον έγκυρο Κατάλογο Διπλωματούχων Ξεναγών-μελών του Σωματείου, ώστε να επιλέξετε τον επόμενο ξεναγό σας

Οι ψηφιακές υπηρεσίες προσφέρονται δωρεάν και απαιτούν να δημιουργήσετε έναν προσωπικό λογαριασμό. Ο λογαριασμός σας αυτός θα εξυπηρετεί όλες τις επαφές σας με τις ψηφιακές υπηρεσίες του σωματείου από όποιο σημείο του κόσμου βρίσκεστε.
